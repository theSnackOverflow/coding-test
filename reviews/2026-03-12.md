## Python 핵심 패턴

### 1. 빠른 입력

```python
import sys
input = sys.stdin.readline
```

`sys.stdin.readline`은 운영체제 버퍼에서 직접 읽기 때문에 기본 `input()`보다 빠르다. 줄 끝에 `\n`을 포함하므로 문자열 처리 전 `.strip()` 필수.

### 2. 다중 값 파싱

```python
a, b = map(int, input().split())        # 두 값
nums = list(map(int, input().split()))  # 여러 값을 리스트로
```

`map(int, iterable)`은 각 원소에 `int()`를 적용하는 지연 평가 객체를 반환한다. 인덱스 접근이 필요하면 `list()`로 변환한다.

### 3. EOF 처리

**방식 1 — readline 빈 문자열 체크**
```python
while True:
    line = input()
    if not line:   # EOF에서 "" 반환 → False
        break
```

**방식 2 — sys.stdin 순회**
```python
for line in sys.stdin:
    print(line, end="")  # line에 \n 포함이므로 end="" 필수
```

### 4. f-string 포맷

```python
print(f"Case #{idx+1}: {a} + {b} = {a+b}")
print(f"{result:.2f}")   # 소수점 2자리 고정
```

### 5. divmod

```python
q, r = divmod(s, 60)   # (몫, 나머지)를 동시에 반환
# s // 60, s % 60 을 한 번에 처리
```

#### 2557 - Hello World
- URL: https://www.acmicpc.net/problem/2557
- 티어: Bronze V

**문제 핵심**

아무 입력 없이 `Hello World!`를 출력한다. 느낌표와 대소문자를 정확히 맞춰야 한다.

**기본 풀이**
```python
print("Hello World!")
```

**다른 풀이 — sys.stdout.write**
```python
import sys
sys.stdout.write("Hello World!\n")
```
`sys.stdout.write()`는 `print()`보다 낮은 수준의 출력 함수다. 개행 문자를 직접 제어해야 한다. 이 문제에서는 차이가 없지만, 대량 출력 시 `print()`보다 빠르다.

**파이썬다운 풀이**

`print("Hello World!")`가 이미 가장 Pythonic하다. 한 줄이면서 의도가 명확하다.

#### 1000 - A+B
- URL: https://www.acmicpc.net/problem/1000
- 티어: Bronze V

**문제 핵심**

한 줄에 공백으로 구분된 두 정수를 읽어 합을 출력한다.

**기본 풀이**
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())
print(a + b)
```

**다른 풀이 — 리스트 합산**
```python
import sys
nums = list(map(int, sys.stdin.readline().split()))
print(nums[0] + nums[1])
```
변수명 없이 리스트 인덱스로 접근하는 방식이다.

**유명한 풀이 — sys.stdout.write**
```python
import sys
input = sys.stdin.readline
output = sys.stdout.write

a, b = map(int, input().split())
output(str(a + b) + "\n")
```
입출력이 매우 많은 문제에서는 `print()` 대신 `sys.stdout.write()`를 사용하면 속도가 향상된다. 단, 반드시 `str()`로 변환하고 `"\n"`을 직접 추가해야 한다.

**파이썬다운 풀이**
```python
import sys
print(sum(map(int, sys.stdin.readline().split())))
```
`sum()`에 `map` 객체를 바로 전달할 수 있다. 중간 변수 없이 한 줄로 처리한다.

#### 1008 - A/B
- URL: https://www.acmicpc.net/problem/1008
- 티어: Bronze V

**문제 핵심**

두 정수 A, B가 주어질 때 A/B를 출력한다. 절대/상대 오차가 10^-9 이하면 정답이다.

**기본 풀이**
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())
print(a / b)
```

**다른 풀이 — 소수점 자리수 명시**
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())
print(f"{a / b:.10f}")
```
오차 허용 범위가 10^-9이므로 소수점 10자리까지 출력하면 확실히 통과된다. 기본 `print(a/b)`도 통과되지만 명시적인 버전이다.

**유명한 풀이 — Decimal 모듈**
```python
from decimal import Decimal, getcontext
getcontext().prec = 50

a, b = map(int, input().split())
print(Decimal(a) / Decimal(b))
```
부동소수점 오차가 없는 정확한 계산이 필요할 때 `decimal.Decimal`을 사용한다. 이 문제에서는 불필요하지만, 오차에 민감한 문제에서 사용하는 유명한 패턴이다.

**파이썬다운 풀이**

`print(a / b)`가 이미 가장 간결하고 Pythonic하다. Python의 `/`는 항상 float을 반환하므로 별도 변환이 필요 없다.

#### 2558 - A+B - 2
- URL: https://www.acmicpc.net/problem/2558
- 티어: Bronze V

**문제 핵심**

두 수가 각각 다른 줄에 입력된다.

**기본 풀이**
```python
import sys
input = sys.stdin.readline

a = int(input())
b = int(input())
print(a + b)
```

**다른 풀이 — sys.stdin 읽기**
```python
import sys
a, b = map(int, sys.stdin.read().split())
print(a + b)
```

**파이썬다운 풀이**

```python
import sys
print(sum(int(line) for line in sys.stdin))
```
`sys.stdin`을 제너레이터로 순회하며 각 줄을 정수로 변환하고 합산한다. 줄 수에 상관없이 동작하는 범용적인 코드다.

#### 10869 - 사칙연산
- URL: https://www.acmicpc.net/problem/10869
- 티어: Bronze V

**문제 핵심**

두 정수의 사칙연산 결과를 각각 다른 줄에 출력한다. 정수 나눗셈(`//`)과 나머지(`%`)의 동작을 이해해야 한다.

**기본 풀이**
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())
print(a + b)
print(a - b)
print(a * b)
print(a // b)
print(a % b)
```

**다른 풀이 — divmod 활용**
```python
import sys
input = sys.stdin.readline

a, b = map(int, input().split())
q, r = divmod(a, b)   # (a // b, a % b) 동시에 계산

print(a + b)
print(a - b)
print(a * b)
print(q)
print(r)
```
`divmod(a, b)`는 몫과 나머지를 튜플로 동시에 반환한다. 성능상 `//`와 `%`를 따로 계산하는 것과 동일하지만, 의미적으로 "나눗셈 결과 전체"를 한 번에 얻는다는 점이 명확하다.

**파이썬다운 풀이**
```python
import sys
a, b = map(int, sys.stdin.readline().split())
results = [a+b, a-b, a*b, a//b, a%b]
print(*results, sep='\n')
```
결과를 리스트에 모은 뒤 `print(*list, sep='\n')`으로 한 번에 출력한다. `*`는 리스트를 언패킹하여 개별 인수로 전달한다.

#### 10950 - A+B - 3
- URL: https://www.acmicpc.net/problem/10950
- 티어: Bronze V

**문제 핵심**

첫 줄에 T, 이후 T줄에 두 수 씩 입력된다.

**기본 풀이**
```python
import sys
input = sys.stdin.readline

n = int(input())
for _ in range(n):
    a, b = map(int, input().split())
    print(a + b)
```

**다른 풀이 — 결과를 모아서 한 번에 출력**
```python
import sys
input = sys.stdin.readline

n = int(input())
results = []
for _ in range(n):
    a, b = map(int, input().split())
    results.append(a + b)

sys.stdout.write('\n'.join(map(str, results)) + '\n')
```
매 반복마다 `print()`를 호출하는 대신, 결과를 리스트에 저장하고 마지막에 한 번에 출력한다. `print()` 호출 횟수를 줄이면 I/O 오버헤드가 감소한다. T가 클 때 유효하다.

**파이썬다운 풀이**
```python
import sys
input = sys.stdin.readline

n = int(input())
print('\n'.join(str(sum(map(int, input().split()))) for _ in range(n)))
```
제너레이터 표현식으로 각 줄을 처리하고 `'\n'.join()`으로 한 번에 출력한다.

#### 10951 - A+B - 4
- URL: https://www.acmicpc.net/problem/10951
- 티어: Bronze V

**문제 핵심**

입력 개수를 알 수 없다. EOF까지 각 줄의 두 수 합을 출력한다.

**기본 풀이**
```python
import sys
input = sys.stdin.readline

lines = []
while True:
    line = input()
    if not line:
        break
    lines.append(line.strip())

for line in lines:
    a, b = map(int, line.split())
    print(a + b)
```

**다른 풀이 — try/except EOFError**
```python
while True:
    try:
        a, b = map(int, input().split())
        print(a + b)
    except EOFError:
        break
```
기본 `input()`은 EOF에서 `EOFError` 예외를 발생시킨다. 예외 처리로 EOF를 감지하는 방식이다. `sys.stdin.readline` 없이 쓸 수 있다는 장점이 있지만, 예외를 제어 흐름으로 쓰는 것은 Python에서 anti-pattern으로 여겨진다.

**유명한 풀이 — for line in sys.stdin**
```python
import sys

for line in sys.stdin:
    a, b = map(int, line.split())
    print(a + b)
```
`sys.stdin`을 파일처럼 순회하는 방식이 가장 간결하고 Pythonic하다. EOF에서 자동으로 반복이 끝난다. 경쟁 프로그래밍에서 가장 널리 사용되는 EOF 처리 패턴이다.

**파이썬다운 풀이**
```python
import sys

results = [sum(map(int, line.split())) for line in sys.stdin]
print('\n'.join(map(str, results)))
```
리스트 컴프리헨션으로 전체 입력을 한 번에 처리하고 `join`으로 출력한다.

#### 10952 - A+B - 5
- URL: https://www.acmicpc.net/problem/10952
- 티어: Bronze V

**문제 핵심**

`0 0` 입력이 종료 신호다. 해당 줄은 출력하지 않는다.

**기본 풀이**
```python
import sys
input = sys.stdin.readline

lines = []
while True:
    line = input()
    if not line:
        break
    lines.append(line.strip())

for line in lines:
    a, b = map(int, line.split())
    if a + b != 0:
        print(a + b)
```

**다른 풀이 — 읽으면서 바로 처리**
```python
import sys

for line in sys.stdin:
    a, b = map(int, line.split())
    if a == 0 and b == 0:
        break
    print(a + b)
```
`0 0`을 만나는 즉시 `break`로 반복을 종료한다. `a + b != 0` 조건보다 명시적이다. (`a=5, b=-5`이면 합이 0이어도 출력해야 하는 문제가 있을 수 있으므로 두 조건을 따로 확인하는 것이 안전하다.)

**파이썬다운 풀이 — walrus 연산자 (Python 3.8+)**
```python
import sys
input = sys.stdin.readline

while (line := input().strip()) and line != '0 0':
    a, b = map(int, line.split())
    print(a + b)
```
walrus 연산자(`:=`)는 표현식에서 변수에 값을 할당한다. `while` 조건 안에서 입력을 받으면서 종료 조건도 동시에 확인한다.

```python
input = sys.stdin.readline

n = int(input())
```

#### 11021 - A+B - 7
- URL: https://www.acmicpc.net/problem/11021
- 티어: Bronze V

**문제 핵심**

`Case #번호: 합` 형식으로 출력한다. 번호는 1부터 시작한다.

**기본 풀이**
```python
import sys
input = sys.stdin.readline

n = int(input())
for idx in range(n):
    a, b = map(int, input().split())
    print(f"Case #{idx+1}: {a+b}")
```

**다른 풀이 — enumerate 활용**
```python
import sys
input = sys.stdin.readline

n = int(input())
lines = [input() for _ in range(n)]

for idx, line in enumerate(lines, 1):
    a, b = map(int, line.split())
    print(f"Case #{idx}: {a+b}")
```
`enumerate(iterable, start=1)`은 1부터 시작하는 인덱스와 값을 동시에 제공한다. `idx+1` 계산이 필요 없어진다.

**파이썬다운 풀이**
```python
import sys
input = sys.stdin.readline

n = int(input())
print('\n'.join(
    f"Case #{i}: {sum(map(int, input().split()))}"
    for i in range(1, n+1)
))
```
`range(1, n+1)`로 1-based 인덱스를 직접 생성하여 `idx+1` 계산을 없앤다.

#### 11718 - 그대로 출력하기
- URL: https://www.acmicpc.net/problem/11718
- 티어: Bronze V

**문제 핵심**

빈 줄이 아닌 줄만 그대로 출력한다.

**기본 풀이**
```python
import sys
input = sys.stdin.readline

while True:
    line = input().strip()
    if not line:
        break
    print(line)
```

**다른 풀이 — fileinput 모듈**
```python
import fileinput

for line in fileinput.input():
    stripped = line.rstrip('\n')
    if not stripped:
        break
    print(stripped)
```
`fileinput`은 파일과 stdin을 통합적으로 다루는 모듈이다. 경쟁 프로그래밍보다 스크립트 작성에서 더 자주 사용된다.

**파이썬다운 풀이**
```python
import sys

lines = sys.stdin.read().split('\n')
result = []
for line in lines:
    if not line:
        break
```

- URL: https://www.acmicpc.net/problem/10809
- 티어: Bronze V

**문제 핵심**

a~z 각각이 문자열에서 처음 등장하는 인덱스를 출력한다. 없으면 -1.

**기본 풀이**
```python

eng = "abcdefghijklmnopqrstuvwxyz"

for c in eng:
    if c in s:
        print(s.index(c), end=" ")
    else:
        print(-1, end=" ")
```

**다른 풀이 — 딕셔너리로 먼저 인덱스 저장**
```python
import sys
input = sys.stdin.readline

s = input().strip()
index_map = {}

for i, c in enumerate(s):
    if c not in index_map:          # 처음 등장한 경우만 저장
        index_map[c] = i

result = [index_map.get(chr(ord('a') + i), -1) for i in range(26)]
print(*result)
```
문자열을 한 번만 순회하여 각 문자의 첫 등장 인덱스를 딕셔너리에 저장한다. O(N+26) 으로 기본 풀이의 O(26N)보다 효율적이다. 단, 이 문제 크기에서는 차이가 없다.

**유명한 풀이 — ord() 활용**
```python
import sys
input = sys.stdin.readline

s = input().strip()
result = [-1] * 26

for i, c in enumerate(s):
    idx = ord(c) - ord('a')     # 'a'=0, 'b'=1, ..., 'z'=25
    if result[idx] == -1:
        result[idx] = i

print(*result)
```
`ord(c) - ord('a')`로 문자를 0~25 정수 인덱스로 변환한다. 알파벳을 배열 인덱스로 사용하는 것은 경쟁 프로그래밍의 고전적 기법이다.

**파이썬다운 풀이**
```python
import sys
input = sys.stdin.readline

s = input().strip()
print(*[s.index(c) if c in s else -1 for c in 'abcdefghijklmnopqrstuvwxyz'])
```
리스트 컴프리헨션과 삼항 연산자로 한 줄로 처리한다. `print(*list)`로 리스트를 공백 구분 출력한다.

```python

import sys
```

```python
a, b = map(int, input().split())
# 올림 나눗셈: ceil(x / a) == b
# x의 최솟값: a*(b-1) + 1

```

```
⟺ k-1 < x/n ≤ k
⟺ n(k-1) < x ≤ nk
⟺ x의 최솟값 = n(k-1) + 1, 최댓값 = nk
```

```python
import sys
input = sys.stdin.readline

hours, mins = map(int, input().split())
time = int(input())

total = hours * 60 + mins + time   # 모든 것을 분 단위로 통합
result_h = (total // 60) % 24
result_m = total % 60

print(result_h, result_m)
```

```python
import sys
nums = list(map(int, sys.stdin.read().split()))[1:]   # 첫 번째 값(N)을 버리고 나머지
print(min(nums), max(nums))
```
`sys.stdin.read()`로 전체를 읽고 `split()`하면 N과 숫자들이 한 리스트에 들어온다. `[1:]`로 N을 제외한 숫자만 추출한다.

#### 2562 - 최댓값
- URL: https://www.acmicpc.net/problem/2562
- 티어: Bronze V

**문제 핵심**

9개의 정수 중 최댓값과 그 위치(1-based)를 출력한다.

**기본 풀이**
```python
import sys

nums = []
for line in sys.stdin:
    nums.append(int(line))

max_val = max(nums)
print(max_val)
print(nums.index(max_val) + 1)
```

**다른 풀이 — enumerate 활용**
```python
import sys

nums = [int(line) for line in sys.stdin]

max_idx, max_val = 0, nums[0]
for i, v in enumerate(nums):
    if v > max_val:
        max_val = v
        max_idx = i

print(max_val)
print(max_idx + 1)
```

**파이썬다운 풀이 — max + key**
```python
import sys

nums = [int(line) for line in sys.stdin]
max_pos, max_val = max(enumerate(nums, 1), key=lambda x: x[1])

print(max_val)
print(max_pos)
```
`enumerate(nums, 1)`은 (1, 첫값), (2, 둘째값), ... 형태의 튜플을 생성한다. `max()`에 `key=lambda x: x[1]`을 주면 값 기준으로 최댓값 튜플을 찾는다. 한 번의 `max()` 호출로 위치와 값을 동시에 구한다.

#### 5355 - 화성 수학
- URL: https://www.acmicpc.net/problem/5355
- 티어: Bronze II

**문제 핵심**

`@`(×3), `%`(+5), `#`(-7) 연산자를 순서대로 적용한다.

**기본 풀이**
```python
import sys
input = sys.stdin.readline

for idx in range(int(input())):
    line = input().split()
    result = float(line[0])

    for op in line[1:]:
        if op == "@":
            result *= 3
        if op == "%":
            result += 5
        if op == "#":
            result -= 7

    print(f"{result:.2f}")
```

**다른 풀이 — 딕셔너리 + lambda**
```python
import sys
input = sys.stdin.readline

ops = {
    "@": lambda x: x * 3,
    "%": lambda x: x + 5,
    "#": lambda x: x - 7,
}

for _ in range(int(input())):
    tokens = input().split()
    result = float(tokens[0])
    for op in tokens[1:]:
        result = ops[op](result)
    print(f"{result:.2f}")
```
연산자와 함수를 딕셔너리로 매핑한다. `if/elif` 체인 대신 딕셔너리 룩업으로 처리한다. 연산자 종류가 많아질수록 이 방식이 유리하다.

**유명한 풀이 — functools.reduce**
```python
import sys
from functools import reduce
input = sys.stdin.readline

ops = {"@": lambda x: x * 3, "%": lambda x: x + 5, "#": lambda x: x - 7}

for _ in range(int(input())):
    tokens = input().split()
    result = reduce(lambda acc, op: ops[op](acc), tokens[1:], float(tokens[0]))
    print(f"{result:.2f}")
```
`reduce(함수, iterable, 초깃값)`은 초깃값부터 시작하여 iterable의 각 원소에 함수를 순서대로 적용한다. 함수형 프로그래밍의 핵심 패턴이다.

#### 3273 - 두 수의 합 (Silver V)
- URL: https://www.acmicpc.net/problem/3273
- 티어: Silver V

**문제 핵심**

서로 다른 양의 정수 N개에서 두 수의 합이 X인 쌍의 개수를 구한다.

**기본 풀이 — 투 포인터 O(N log N)**
```python
import sys
input = sys.stdin.readline

n = int(input())
nums = list(map(int, input().split()))
target = int(input())

nums.sort()

answer = 0
left, right = 0, n - 1

while left < right:
    s = nums[left] + nums[right]

    if s == target:
        answer += 1
        left += 1
        right -= 1
    elif s < target:
        left += 1
    else:
        right -= 1

print(answer)
```

**다른 풀이 — 완전 탐색 O(N²)**
```python
import sys
input = sys.stdin.readline

n = int(input())
nums = list(map(int, input().split()))
target = int(input())

count = 0
for i in range(n):
    for j in range(i + 1, n):
        if nums[i] + nums[j] == target:
            count += 1

print(count)
```
모든 쌍을 검사한다. N이 최대 100,000이면 10^10 연산으로 시간 초과. N이 작은 경우(수천 이하)에만 사용 가능하다.

**유명한 풀이 — 해시셋 O(N)**
```python
import sys
input = sys.stdin.readline

n = int(input())
nums = list(map(int, input().split()))
target = int(input())

seen = set()
count = 0

for x in nums:
    complement = target - x
    if complement in seen:
        count += 1
    seen.add(x)
```
"Two Sum" 문제의 고전적인 O(N) 해시셋 풀이다. 각 숫자를 순회하며, `target - x`(보수)가 이미 처리된 숫자 집합에 있으면 쌍이다. 정렬이 필요 없어 실제로 가장 빠르다.

단, 이 문제에서는 숫자가 서로 다르기 때문에 같은 값이 두 번 쓰이는 경우를 걱정할 필요가 없다.

**파이썬다운 풀이 — set 컴프리헨션**
```python
import sys
input = sys.stdin.readline

n = int(input())
nums = list(map(int, input().split()))
target = int(input())

num_set = set(nums)
# 각 x에 대해 target-x가 집합에 있고, x != target-x인 경우
# 중복 카운트를 피하기 위해 x < target-x 조건 추가
count = sum(1 for x in nums if (target - x) in num_set and x < target - x)
print(count)
```
`x < target - x` 조건으로 각 쌍을 한 번만 카운트한다. `x == target - x`인 경우(같은 값 두 개)는 이 문제에서 모든 수가 서로 다르므로 발생하지 않는다.

**투 포인터 알고리즘 동작 추적 예시**

```
nums = [5, 3, 4, 2, 1], target = 5

정렬: [1, 2, 3, 4, 5]
        L           R

step 1: 1+5=6 > 5 → right--
        L        R
step 2: 1+4=5 == 5 → count=1, left++, right--
           L     R
step 3: 2+4=6 > 5 ... 잠깐, 4는 이미 right--로 지나쳐서 R=2
           L  R
step 3 실제: 2+3=5 == 5 → count=2, left++, right--
              LR  (left >= right → 종료)

결과: 2쌍
```

## 자주 실수하는 부분

### 1. readline 개행 문자 미처리

```python
s = input()        # "hello\n"
s = input().strip()  # "hello"
```

`sys.stdin.readline`은 `\n`을 포함한다. 숫자 파싱(`map(int, ...split())`)에서는 자동 처리되지만, 문자열 비교나 인덱싱 전에는 `.strip()` 필수.

### 2. 0-based vs 1-based 인덱스

```python
nums.index(max_val) + 1   # Python 인덱스는 0-based, 문제는 1-based
for idx in range(n):
    print(f"Case #{idx+1}: ...")
for i, v in enumerate(nums, 1):   # enumerate(start=1)로 1-based 직접 생성
    ...
```

### 3. 정수 나눗셈 vs 실수 나눗셈

```python
10 // 3   # 3  (정수 나눗셈, 몫만)
10 / 3    # 3.333...  (실수 나눗셈)
```

### 4. EOF 처리 상황별 정리

| 상황 | 방법 |
|------|------|
| EOF까지 줄별 파싱 | `for line in sys.stdin` |
| EOF까지 전체 읽기 | `sys.stdin.read()` |
| 종료 마커(`0 0`) | 값 조건 확인 후 `break` |

### 5. 시간 계산 — 총 단위 변환법이 더 안전하다

```python
# 단계적 올림 (실수 가능성 있음)
s += c
m += s // 60; s %= 60
h += m // 60; m %= 60

# 총 초로 변환 후 역산 (더 안전)
total = h*3600 + m*60 + s + c
h, rem = divmod(total, 3600)
m, s = divmod(rem, 60)
h %= 24
```
